# Delta Scientific Calculator 🧮

A lightweight, powerful, and fully responsive Scientific Calculator built as a [WebXDC](https://webxdc.org/) application for Delta Chat. It handles complex engineering and mathematical expressions with high precision, specifically optimized to process trigonometric functions in **Degrees**.

## ✨ Features

*   **Advanced Scientific Operations:** Supports trigonometric functions (sin, cos, tan) and their inverses (asin, acos, atan), logarithms, roots, and powers.
*   **Degree-Based Trigonometry:** Unlike native JavaScript math functions which default to radians, this calculator automatically and accurately computes all trigonometric expressions in degrees.
*   **Perfect Order of Operations:** Flawlessly evaluates long, mixed expressions strictly following standard mathematical precedence (BODMAS/PEMDAS).
*   **Smart Display UI:** Automatically scales down font sizes for long equations to prevent screen overflow, ensuring a consistently clean look.
*   **History Logs:** Keeps track of your previous calculations for easy reference.
*   **WebXDC Sharing:** Share your calculation results directly into the Delta Chat chatroom as text or rendered images.
*   **Clean Interface (Shift Toggle):** Uses a `Shift` button to reveal inverse and secondary functions without cluttering the main keypad.
